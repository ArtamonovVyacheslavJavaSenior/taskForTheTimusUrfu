# Timus 1000 — A+B Problem

###  Условие
Вычислить сумму двух целых чисел **a** и **b**.

**Ограничения:**
-  Время: 1 секунда
-  Память: 64 МБ

---

##  Входные данные
Два целых числа: a b

## Выходные данные
Одно число — сумма:
a + b 

Результат
a + b

## Замечание
Используйте оператор сложения `+`.
