# Голосование по группам на острове

## Описание задачи

В одном из островных государств Карибского бассейна все решения принимались простым большинством голосов на общем собрании. С ростом населения было введено реформированное голосование:

1. Все избиратели делятся на **K групп** (необязательно равные по численности).
2. Голосование проводится отдельно в каждой группе:
    - Группа голосует «за», если более половины членов группы голосует «за».
    - В противном случае группа голосует «против».
3. Решение принимается положительно, если **более половины групп** проголосовали «за».

Сторонники партии могут распределять своих людей по группам так, чтобы принимать решения меньшим числом голосов, чем требуется при общем голосовании.

**Задача:** определить минимальное количество сторонников партии, необходимое для принятия решения, при оптимальном распределении по группам.

---

## Входные данные

1. Первая строка: нечётное число **K** — количество групп (1 ≤ K ≤ 101).
2. Вторая строка: **K нечётных чисел**, задающих количество избирателей в группах.
    - Общее население острова ≤ 9999 человек.



*Пояснение:* В этом примере партии достаточно иметь по 3 сторонника в первых двух группах и 4 в третьей, всего 10, но минимальное — 9.

---

## Замечания

- Все числа в входе нечётные.
- Для каждой группы минимальное количество «за» голосов: `(size + 1)/2`.
- Для принятия решения нужно более половины групп.
- Оптимально выбираются группы с наименьшими необходимыми «за» голосами, чтобы минимизировать общее число сторонников.
