# Timus 1002 — Телефонные номера

## Описание задачи
В современном мире телефонные номера становятся длиннее, и чтобы их проще запоминать, используется сопоставление цифр буквам. Каждой цифре соответствует набор букв:

1: i j 

2: a b c 

3: d e f

4: g h 

5: k l 

6: m n

7: p r s 

8: t u v 

9: w x y

0: o q z

Задача состоит в том, чтобы найти **самую короткую последовательность слов** из словаря, которая соответствует заданному телефонному номеру.

- Если несколько последовательностей имеют одинаковую минимальную длину, можно выбрать любую.
- Если решения нет, выводится: `"No solution."`.

---

## Входные данные
Ввод состоит из нескольких тестов. Каждый тест имеет следующий формат:

1. Первая строка: номер телефона (не более 100 цифр).
2. Вторая строка: количество слов в словаре (максимум 50 000).
3. Последующие строки: слова словаря (каждое ≤ 50 строчных букв).
4. Последняя строка ввода содержит число `-1`, которое обозначает конец ввода.

**Ограничения:**
- Общий размер входного файла ≤ 300 КБ.

---
