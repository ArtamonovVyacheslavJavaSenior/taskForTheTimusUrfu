# Timus 1001 — Обратный корень

## Описание задачи
Входной поток содержит набор целых чисел Ai, разделённых произвольным количеством пробелов и переводов строк.  
Каждое число удовлетворяет условию: 

0 ≤ Ai ≤ 10 ^ 18

Общий размер входного файла — не более 256 КБ.

Требуется для каждого числа Ai, начиная с последнего и заканчивая первым, вывести квадратный корень числа.  
Точность вывода — не менее четырёх знаков после десятичной точки.  
Каждое значение должно выводиться в отдельной строке.
