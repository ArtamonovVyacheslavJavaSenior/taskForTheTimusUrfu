# Факториалы!!! 1083.


## Описание задачи

#### Определение 1 
n!!…! = n(n−k)(n−2k)…(n mod k), если n не делится на k; n!!…! = n(n−k)(n−2k)…k, 
если n делится на k (знаков ! в обоих случаях k штук).

#### Определение 2
X mod Y — остаток от деления X на Y.
Например, 10 mod 3 = 1; 3! = 3·2·1; 10!!! = 10·7·4·1.


### Исходные данные

В единственной строке сначала дано целое число n, 1 ≤ n ≤ 10, 
затем ровно один пробел, затем k восклицательных знаков, 1 ≤ k ≤ 20.

### Результат

Выведите одно число — n!!…!
